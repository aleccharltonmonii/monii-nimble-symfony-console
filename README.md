Requirements
------------

 * PHP 5.5+


Installation
------------

```bash
$> composer require monii/nimble-symfony-console
```

Until a stable version has been released or if a development version is preferred, use:

```bash
$> composer require monii/nimble-symfony-console:@dev
```
